# Disponibilité du nom App Store — « Standup Timer »

**Date de recherche :** 2026-06-30
**Question :** le nom exact « Standup Timer » est-il libre sur l'App Store (iOS / Mac) ?

## Conclusion (TL;DR)

Le nom exact **« Standup Timer » est DÉJÀ PRIS** sur l'App Store.
App Store Connect impose un **nom d'app unique** au niveau mondial : on ne pourra
**pas** publier sous le nom nu « Standup Timer ». Il faut **un nom qualifié**
(avec un mot de marque / sous-titre distinctif). De plus, plusieurs apps voisines
occupent le champ sémantique « standup / scrum timer », ce qui plaide pour un nom
plus différenciant côté ASO (référencement App Store) comme côté marque.

## Recommandation

1. **Ne pas compter sur le nom nu « Standup Timer »** dans App Store Connect.
2. Choisir un **nom de marque court + descripteur**. Le champ « App Name » (30
   caractères) porte la marque ; le **sous-titre** (30 car.) porte les mots-clés
   ASO (« standup », « scrum », « timer », « daily »). On peut donc garder
   « standup timer » dans le sous-titre / mots-clés sans avoir le nom exact.
3. Vérifier la disponibilité finale **directement dans App Store Connect** au
   moment de réserver le nom (seul juge fiable de l'unicité), et idéalement la
   dispo du domaine `.app` correspondant.

## Noms de repli proposés (3–5)

Tous gardent la lisibilité « timer de standup » tout en étant distinctifs :

1. **Standup Signal** — s'appuie sur le concept maison « le Signal » (anneau
   vert→rouge) déjà au cœur du produit et de la landing page. Fort, ownable, pas
   un terme générique. *(Recommandé n°1.)*
2. **StandTime** — contraction « Stand(up) + Time », courte, marque facile à
   déposer, domaine probablement plus accessible.
3. **Daily Ring** — référence à l'anneau du Signal + « daily » (le rituel).
   Court, mémorisable, oriente « réunion » plutôt que « posture ».
4. **Tempo Standup** — « Tempo » porte l'idée de cadence/rythme ; le mot
   « standup » reste présent pour l'ASO.
5. **Roundtable Timer** — « roundtable » (tour de table) décrit le format du
   standup ; nom descriptif mais plus distinctif que « Standup Timer ».

Format App Store conseillé, ex. n°1 :
- **App Name :** `Standup Signal`
- **Subtitle :** `Timer de daily standup & scrum`
- (mots-clés : standup, scrum, daily, timer, meeting, agile, équipe…)
